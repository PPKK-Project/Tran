## 🌟 프로젝트 API 명세서 (정리본)

### I. 사용자 및 인증 관리 (User & Auth)

| 목표 | 메소드 | 엔드포인트 | 설명 |
| :--- | :--- | :--- | :--- |
| **회원가입** | `POST` | `/auth/signup` | 신규 사용자 계정을 생성합니다. |
| **로그인** | `POST` | `/auth/login` | 사용자 인증 후 JWT 토큰을 발급합니다. |
| **로그아웃** | `POST` | `/auth/logout` | 서버 측 토큰 무효화(선택적) 또는 클라이언트 측 토큰 삭제 유도 |
| **내 정보 조회** | `GET` | `/users/me` | 현재 로그인된 사용자의 정보를 조회합니다. |
| **내 정보 수정** | `PATCH` | `/users/me` | 사용자 정보를 수정합니다. |
